# Emoji Platform

一个长期维护的全球 Emoji 平台项目。

## 项目目标

- Emoji 搜索
- Emoji 复制
- Emoji 含义解释
- Unicode 信息查询
- Emoji 分类浏览
- Emoji 专题页
- 多语言 SEO
- 后台 CMS 管理
- 搜索统计
- 后续支持海内外用户访问

## 当前阶段

**Phase 0** - 项目保护机制与远程仓库初始化。

## 技术栈（规划）

- pnpm monorepo
- Next.js + TypeScript + Tailwind CSS（前台 & 后台）
- NestJS + TypeScript（API）
- PostgreSQL + Prisma
- Redis
- Meilisearch
- Docker Compose
